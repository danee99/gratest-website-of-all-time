Taschenrechner

Hier siehst du einen einfachen Taschenrechner. Probier ihn einfach mal aus.

Erklärung

Der Taschenrechner besteht aus einem Textfeld, das auf "readonly" gesetzt ist. Hier werden sowohl die Benutzereingaben als auch das Endergebnis stehen. Dann gibt es verschiedene Buttons. Zum einen für die Zahlen von 0 bis 9 und zum anderen für die Rechen Operatoren (d.h. Plus, Minus, etc.). Und schließlich gibt es zwei Buttons, einmal zum Zurücksetzen der Eingabe ("C") und zum Berechnen der Eingabe ("="). Die JavaScript-Funktionen sehen wie folgt aus. Wir verwenden hier die eval()-Funktion, die einen String (z.B: "2*3+4") berechnen kann, also sehr praktisch. 

                    
    function addToInput(x){
        document.getElementById("calculator-result").value += x;
    }
    
    function clearInput(){
        document.getElementById("calculator-result").value = "";
    }
    
    function calculate(){
        document.getElementById("calculator-result").value 
        = eval(document.getElementById("calculator-result").value);
    }
